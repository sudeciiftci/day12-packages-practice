# Java Packages Practice

This project demonstrates how **packages and imports** work in Java.

## Structure

### school package
- Contains the `packages` class.
- Provides a simple `show()` method that prints a message.

### main package
- Contains the `packages2` class.
- Imports the `school.packages` class and uses it.

## Key Concepts
- Creating packages in Java
- Using the `package` keyword
- Importing classes from other packages
- Organizing code into separate modules

## Purpose
To understand:
- How Java projects are structured using packages
- How to access classes from different packages
- Code organization and reusability
